Joe Rock'n Trad

Joe Rock'n Trad is a small music festival organized by the Joensuu Conservatory. In this project I designed a new visual identity for the event and built a new website.

About the event

As the name suggests, Joe Rock'n Trad features bands that play rock and folk music. The festival consists of multiple concerts held at various locations across Joensuu.

About the project

The project started with designing the event poster. I presented the client with three different concepts, and we moved forward with the floral guitar theme that best captured the festival's spirit. This theme was then carried over to other design elements, creating a cohesive and recognizable visual identity for the event.

My tasks

I created a simple yet functional website for the event using Adobe Dreamweaver. The site provides information about the performers and the concert schedule, making it easy for attendees to find key details.

In addition to designing and building the website, I also designed the official festival poster, a customizable poster template for individual concerts, a Facebook banner, and other promotional materials. These designs helped create a consistent visual identity across different platforms and marketing channels, reinforcing the festival's brand and making it easily recognizable.
